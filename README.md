# 🚖 Uber Ride Analysis Dashboard

## 📌 Project Overview

This interactive Power BI dashboard analyzes Uber ride operations to provide meaningful insights into booking performance, ride cancellations, revenue, customer ratings, and vehicle performance.

The dashboard was built to transform raw operational data into clear, interactive business insights that support better decision-making.

---

## 🎯 Business Objectives

* Analyze booking performance
* Monitor ride cancellations
* Track booking revenue
* Compare vehicle performance
* Evaluate customer ratings
* Explore ride distance trends

---

## 📊 Dashboard Features

* Interactive Filters
* KPI Cards
* Dynamic Vehicle Selection
* Bookmarks & Pop-up Panel
* Customer Rating Visualization
* Ride Cancellation Analysis
* Revenue Analysis
* Vehicle Type Comparison

---

## 🛠️ Tools & Technologies

* Power BI
* DAX
* Power Query
* Data Modeling

## 📈 Key KPIs

* Total Bookings

* Average Ride Distance
* Average Customer Rating

## 💡 Key Insights

* Booking trends vary across vehicle types.
* Customer cancellations reveal common service issues.
* Driver cancellations help identify operational challenges.
* Customer ratings provide insight into service quality.
* Interactive filters allow detailed business exploration.
